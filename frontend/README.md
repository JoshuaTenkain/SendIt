# SEND-IT Frontend

Next.js frontend for the SEND-IT courier aggregation platform.

## Setup

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- User authentication (signup/login)
- Quote comparison
- Booking management
- Payment integration (PayFast)
- Shipment tracking
- Admin dashboard

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide Icons
